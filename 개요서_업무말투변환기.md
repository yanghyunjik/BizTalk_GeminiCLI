# 업무 말투 변환기 — 프로그램 개요서

> **대상 독자**: 바이브 코딩(Vibe Coding) 입문 개발자
> **실습 형태**: One Day 프로젝트
> **문서 목적**: 오늘 우리가 무엇을, 왜 만드는지 이해하기

---

## 1. 이 프로그램은 무엇인가요?

**업무 말투 변환기**는 하고 싶은 말을 편하게 입력하면,
메시지를 받는 상대방에 맞는 **적절한 업무 언어로 자동 변환**해주는 서비스입니다.

> **예시**
>
> 내가 입력한 말 → *"내일까지 보고서 제출 어려울 것 같음"*
>
> ✅ 상사에게 → *"안녕하세요. 보고서 제출 일정과 관련하여 말씀드립니다. 예상치 못한 사정으로 내일까지 제출이 어려울 것 같습니다. 일정 조율이 가능하신지 여쭤봐도 될까요?"*
>
> ✅ 고객에게 → *"안녕하세요. 보고서 제출 일정이 당초 예정보다 다소 늦어질 것 같습니다. 불편을 드려 대단히 죄송하며, 가능한 빠른 시일 내에 전달드리겠습니다."*

---

## 2. 왜 만드나요?

직장인이라면 누구나 이런 경험이 있습니다.

- 💬 *"이걸 상사한테 어떻게 말해야 하지?"*
- 💬 *"고객한테 보내는 건데 이 표현이 맞나?"*
- 💬 *"내용은 아는데 문장이 어색한 것 같아서 계속 고치고 있어..."*

**핵심 내용은 알고 있지만, 표현 방법을 고민하느라 시간을 낭비하는 경우**가 많습니다.
업무 말투 변환기는 이 고민을 AI가 대신 해결해주는 서비스입니다.

---

## 3. 누가 사용하나요?

| 사용자 | 상황 |
|--------|------|
| 신입 사원 | 상사·고객에게 보낼 메시지 표현이 어려울 때 |
| 실무 담당자 | 빠르게 정중한 업무 메시지를 작성해야 할 때 |
| 비즈니스 커뮤니케이션이 어색한 누구나 | 말은 알겠는데 글로 쓰기 어려울 때 |

---

## 4. 핵심 기능

### 기능은 딱 하나입니다 — **말투 변환**

사용자가 할 일은 두 가지뿐입니다.

1. **전달하고 싶은 내용**을 편하게 입력한다
2. **메시지를 받을 상대**를 선택한다

그러면 AI가 상황에 맞는 업무 언어로 바꿔줍니다.

### 선택 가능한 수신 대상

| 대상 | 변환 스타일 |
|------|------------|
| 👔 상사 / 임원 | 격식 있고 공손한 경어체 |
| 🤝 타팀 동료 | 정중하면서도 협조적인 어조 |
| 🌐 고객 / 외부 | 친절하고 신뢰감을 주는 서비스 어조 |
| 👥 팀 내 동료 | 간결하고 실무적인 어조 |

---

## 5. 어떤 기술로 만드나요?

기술 세부 사항은 PRD 문서에서 다룹니다.
여기서는 전체 구조만 간단히 이해해 주세요.

```
사용자 화면          처리 서버            AI 두뇌
(HTML/CSS/JS)  →  (Python FastAPI)  →  (Upstage Solar-Pro2)
   입력 · 출력        요청 처리             말투 변환
```

- **화면**: 웹 브라우저에서 바로 사용할 수 있는 HTML 페이지
- **서버**: 사용자 요청을 받아 AI에게 전달하는 Python 서버
- **AI**: 한국어에 특화된 Upstage의 Solar-Pro2 언어 모델

---

## 6. 오늘 하루 실습 범위

One Day 과정이므로, **핵심 기능에만 집중**합니다.

### ✅ 오늘 완성할 것

- 텍스트 입력 화면 (HTML)
- 수신 대상 선택 기능
- AI 말투 변환 처리 (FastAPI + Solar-Pro2)
- 변환 결과 화면 출력
- GitHub 업로드 + Vercel 배포

### 🔜 오늘 하지 않는 것 (이후 확장 가능)

- 로그인 / 회원 기능
- 변환 이력 저장
- 디자인 고도화

> 오늘의 목표는 **"완벽한 서비스"가 아니라 "작동하는 서비스"** 를 직접 만들어보는 것입니다.
